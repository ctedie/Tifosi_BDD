\# TIFOSI - Conception d’une base de données MySQL



\## 📌 Contexte



Dans le cadre de la formation Développeur Web / Web Mobile, ce projet consiste à concevoir une base de données relationnelle pour le site du restaurant italien de street-food \*\*Tifosi\*\*.



La base de données est développée sous \*\*MySQL\*\* et hébergée en local.



---



\## 🎯 Objectifs pédagogiques



À travers ce projet, les compétences suivantes sont mobilisées :



\- Recenser les informations d’un domaine métier

\- Organiser les données selon le modèle relationnel

\- Construire l’organisation physique des données

\- Mettre en œuvre les instructions SQL de création et modification

\- Exécuter des requêtes SQL de complexité variable



---
